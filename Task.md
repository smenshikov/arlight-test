# Описание задания

## Описание страницы
Страница состоит из четырёх блоков:
- Шапка — заголовок и счётчик открытых вкладок
- Сетка карточек — 4 карточки с контентом
- Форма подписки — поле email, кнопка, список подписчиков
- Кнопка смены темы — переключает светлую/тёмную тему

## Задание 1. Адаптивная сетка карточек
Сверстайте блок из 4 карточек. Каждая карточка должна содержать:
- Заголовок
- Текстовое описание (3 строки)
- Кнопку

Требования к адаптивности:
- На широких экранах — 4 карточки в ряд
- На планшетах — 2 карточки в ряд
- На телефонах — 1 карточку в ряд

Карточки должны быть одинаковой высоты независимо от количества текста. Кнопка должна располагаться внизу карточки.

## Задание 2. Форма подписки
Под сеткой карточек реализуйте форму подписки на рассылку:
- Поле ввода email
- Кнопка «Подписаться»
- Список уже подписанных email'ов

Логика работы:
- При клике на кнопку проверьте, что введён корректный email (содержит @ и точку после неё)
- Если email некорректный — покажите сообщение об ошибке под полем ввода. Сообщение должно появляться плавно. При повторном вводе в поле ошибка должна исчезать
- Если email корректный — добавьте его в список подписчиков, если его там ещё нет (регистр букв не должен влиять на проверку дубликатов)
- После успешного добавления очистите поле ввода
- Список подписчиков должен обновляться без перезагрузки страницы
- Начальный список подписчиков должен содержать 2–3 примера email'ов

## Задание 3. Shared Worker
Реализуйте синхронизацию между вкладками с помощью Shared Worker.

Что должно синхронизироваться:
- Счётчик открытых вкладок — в шапке страницы отображается количество открытых вкладок с вашим сайтом. При открытии или закрытии вкладки счётчик обновляется на всех вкладках.
- Тема оформления — при клике на кнопку смены темы она переключается одновременно на всех открытых вкладках.

Shared Worker должен быть в отдельном файле.
