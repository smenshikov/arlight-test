# Arlight Test Project

1. Веб-приложение для демонстрации.
2. Доступно по адресу: [https://smenshikov.github.io/arlight-test](https://smenshikov.github.io/arlight-test)

## Обзор проекта

Проект представляет собой одностраничное приложение с адаптивным дизайном, включающее:
- Адаптивную сетку карточек
- Форму подписки с валидацией email
- Синхронизацию между вкладками через Shared Worker
- Переключение темы оформления

## Структура проекта

```
arlight-test/
├── index.html          # Основная HTML-страница
├── script.js           # Основная логика приложения
├── shared-worker.js    # Shared Worker для синхронизации вкладок
├── styles.css          # Стили приложения
├── server.js           # Локальный сервер для разработки
├── package.json        # Зависимости и скрипты проекта
├── Task.md             # Техническое задание
└── README.md           # Этот файл
```

## Ключевые компоненты

### 1. Адаптивная сетка карточек
- Реализована с помощью CSS Grid
- Поддержка 3-х точек адаптивности:
  - Десктоп: 4 карточки в ряд
  - Планшет: 2 карточки в ряд  
  - Мобильные: 1 карточка в ряд
- Карточки одинаковой высоты с кнопками внизу

### 2. Форма подписки
- Валидация email в реальном времени
- Плавная анимация ошибок
- Проверка дубликатов (нечувствительная к регистру)
- Динамическое обновление списка подписчиков

### 3. Shared Worker
- Синхронизация счётчика открытых вкладок
- Общая тема оформления между вкладками
- Обработка подключения/отключения вкладок

### 4. Система тем
- Светлая и тёмная тема
- Сохранение состояния в Shared Worker
- Плавные переходы между темами

## Технологический стек

- **HTML5**: Семантическая разметка
- **CSS3**: Flexbox, Grid, CSS-переменные, медиа-запросы
- **JavaScript (ES6+)**: DOM API
- **Shared Worker API**: Для межтабовой синхронизации
- **Node.js**: Локальный сервер для разработки

## Установка и запуск

1. Клонировать репозиторий:
```bash
git clone <repository-url>
cd arlight-test
```

2. Установить правильную версию Node.js (v20.19.0):

```bash
nvm use
```

3. Установить зависимости:
```bash
npm install
```

4. Запустить локальный сервер:
```bash
npm start
```

5. Открыть приложение в браузере:
```
http://localhost:3000
```

## Особенности реализации

### Shared Worker
- Файл `shared-worker.js` обрабатывает подключения вкладок
- Хранит общее состояние (тема, счётчик вкладок)
- Рассылает обновления всем подключённым вкладкам

### Валидация email
- Проверка наличия символа @ и точки после него
- Нечувствительная к регистру проверка дубликатов
- Плавные CSS-анимации для ошибок

### Адтивность
- Использование CSS Grid для карточек
- Медиа-запросы для разных размеров экрана
- Flexbox для выравнивания элементов внутри карточек

## Тестирование

Для тестирования функциональности Shared Worker:
1. Откройте приложение в нескольких вкладках
2. Наблюдайте за синхронизацией счётчика вкладок
3. Переключайте тему и проверяйте синхронизацию
4. Проверяйте обновление счётчика при закрытии вкладок

## Поддерживаемые браузеры

- Chrome (рекомендуется для тестирования Shared Worker)
- Firefox
- Safari
- Edge

## Заметки для разработчиков

- Shared Worker требует HTTPS или localhost для работы
- CSS-переменные тем определены в `:root`
- Все асинхронные операции обёрнуты в try-catch
